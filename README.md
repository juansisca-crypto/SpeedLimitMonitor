# Speed Limit Monitor

App Android que monitorea tu velocidad en tiempo real y te avisa cuando excedes el límite permitido de la vía.

## Características
- GPS de alta precisión
- Límites de velocidad automáticos vía OpenStreetMap
- Alertas visuales y sonoras
- Funciona en segundo plano

## Descargar APK
Ve a la sección **Actions** de este repositorio y descarga el último APK compilado.

## Instalación Manual
1. Activa "Orígenes desconocidos" en tu Android
2. Instala el APK
3. Concede permisos de ubicación
4. ¡Listo para usar!
